# Исследование надёжности заёмщиков — анализ банковских данных

### Тема обучения
Предобработка данных.

### Задачи проекта
На основе статистики о платёжеспособности клиентов исследовать влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок.

### Описание проекта
Заказчик — кредитный отдел банка. Нужно разобраться, влияет ли семейное положение и количество детей клиента на факт погашения кредита в срок. Входные данные от банка — статистика о платёжеспособности клиентов. Результаты исследования будут учтены при построении модели кредитного скоринга.

### Используемые библиотеки
pandas, seaborn, matplotlib.

### Итог
На основе данных кредитного отдела банка исследовал влияние семейного положения и количества детей на факт погашения кредита в срок. Была получена информация о данных. Определены и обработаны пропуски. Заменены типы данных на соответствующие хранящимся данным. Удалены дубликаты. Категоризованы данные.
